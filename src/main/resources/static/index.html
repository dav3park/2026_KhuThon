Project Flow

협업 프로젝트가 만들어지는 과정

  1. 1 장인과 에셋 탐색

    기술 분야, 작업 이미지, 협업 가능 조건을 확인합니다.

  2. 2 기획안 작성

    상품 콘셉트, 제작 방식, 수량, 일정, 판매 전략을 제안합니다.

  3. 3 프로젝트 진행

    장인과 창작자가 역할을 나누고 제작 과정과 승인 이력을 기록합니다.

  4. 4 작품 판매

    완성된 결과물을 정해진 수량으로 공개하고 재고를 투명하게 보여줍니다.
